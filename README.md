# Extended with tailwindcss, prisma, and prisma modules through npm scripts

## Running Locally
### Clone the repo
```
   $ git clone 
```

### Install deps
```
   $ yarn install 
```
   ### OR
```
   $ npm install
```
First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
